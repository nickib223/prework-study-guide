HTML

  • Meta and head do do not show up for the user, instead they are information for browsers and programmers about the content of the site (with the exception of title which is the title of the page seen in the browser tab)
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use h1-h6 in order to keep pages easily readable. h1 should only be used once.
  • img requires a source (src=) and alt which provides an explanation of the image making it accessible for those using adaptive readers.
  • p starts a new paragraph

CSS

  • A margin indicates how much space we want around the outside of an element.
  • Colors can be stated in semantic terms or by hexadecimal code using the format #number
  • A padding indicates how much space we want around the content inside an element.
  • A period preceding a variable is a class selector meaning any element assigned to the class will gain the assignments of this selector
  • CSS rules contain a selector {property: value; property: value; etc }
  • A property: value pair is known as a declaration. Multiple pairs are known as a declaration block.
  • a needs to be added to the HTML file to create a connection between CSS styling and HTML.

Git

  • git status: checks what branch we are currently on
  • git checkout main: switches to main branch.
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: transfers to using the most current main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Function name() {} defines a function
  • Don't forget to call your function!
  • Math.floor and Math.random can be used for randomization.
  • Arrays are containers for multiple variables. Indices start at 0.
  • for loops will continue to run until the value is no longer truthy (in other words, it is now falsy.)
  • if and else if statements provide rules for the computer to check for until it finds a truthy value.
  • console.log is like print function in python and shows up in the console section of Chrome Dev Tools.
